Topic 2 - Link using an Image

Adding a hyperlink to text is great, but I would like to add a hyperlink to an image. A picture can be worth a thousad words and would be much more useful for sending someone to a website to purchase a product. A good example of this can be found on Viclav Smils' Books webpage.

I believe adding hyperlinks to an image would improve my website in the following ways:
